HTML provides various tags to style and format text, making it more readable and structured.
Commonly Used Text Formatting Tags

5. (Italic)Displays text in italics for stylistic purposes.Example:

This is italic text.

6. (Emphasized Text)Similar to Example:

This is emphasized text.

7. (Citations)Used for citing sources.Example:

The Great Gatsby by F. Scott Fitzgerald.

8. Represents contact information.Example:
Contact: support@example.com
9. (Superscript)Raises text above the normal line.Example:

X2 + Y2 = Z2

10. (Subscript)Lowers text below the normal line.Example:

H2O is the chemical formula for water.

11. Reduces the font size of text.Example:

This is small text.

12. (Deprecated)Increases text size (not recommended in modern HTML).Example:

This is big text.

13. Highlights text.Example:

This is highlighted text.

14. (Deprecated)Strikes through text (use Example:

This is struck-through text.

15. (Deleted Text)Represents deleted text.Example:

This is deleted text.

16. (Underlined Text)Underlines text (use CSS for styling instead).Example:

This is underlined text.

17. (Inserted Text)Represents inserted text.Example:

This is newly added text.

18. (Short Quotations)Used for inline quotes.Example:

She said, HTML is easy to learn.

19. (Teletype/Monospace) (Deprecated)Displays text in a fixed-width font.Example:

This is monospace text.

20. (Code Snippets)Displays text in a monospace font for code.Example:

Use console.log('Hello World'); in JavaScript.
