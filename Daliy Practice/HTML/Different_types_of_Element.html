There are many type of elements :-- 1. Block level Element 2. Inline level Element 3. Semantic Element

Block level Element:- this types of element occupy full width , always start with new line. example:- p, div, hr, section, table, list, form, h1 to h6.

Inline level Element:- this type of element occupy only necessary width, and never start with new line. example:- span, small, sup, sub, storng, button, etc.
