There are Many types of tags:- 1.Block Tag 2. Void Tag 3. Null Tag 4. Empty Tag

Block Tag:- The tag which have both opening and closing tag is called block tag. example:- p, table, div, body, etc.

Void Tag :- The tag which have only opening tag is called void tag. example:- br, hr, image, input, meta, link, etc.

Null Tag:- This tag is not define in HTML but in other programing language null means absence value.

Empty Tag:- The tag means tag without content.
